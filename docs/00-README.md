# MyWave AI-TEAM — Preset Pack v1.1
Дата сборки: 2026-03-04

Это практический пакет пресетов для иерархической multi-agent системы MyWave по модели:
**Конвейер → Круглый стол → Суд (AGM)**, с доставкой итогов владельцу в **Telegram (бот-личка)**.

## Что нового в v1.1
1) Пресеты и маршрутизация **под реальные проекты MyWave** (сайт/боты/медиа/ивенты/игра/инфраструктура/R&D).
2) Формальная схема критичности: **Plan vs Execute**, и **human-in-the-loop** для критических действий.
3) Мэппинг на модель пула **agency-agents (51 агент)**: теги, правила подбора и запреты самопроверки.
4) Единые стандарты артефактов (templates/skills) + единый журнал решений.

## Быстрый старт
1) Открой `docs/02-Operating-System-SOP.md` (процесс и правила).
2) Скопируй `docs/03-AGM-System-Prompt.md` как системный промпт AGM.
3) Проверь `config/routing.yaml` (маршрутизация задач).
4) Подключи папки `cursor/subagents` и `cursor/skills` в Cursor.
5) Для Telegram/серверной сборки см. `src/` и `docker/`.

## Важные файлы
- `docs/01-MyWave-Project-Map.md` — карта проектов и доменов
- `config/routing.yaml` — какие задачи идут в какие конвейеры/круглые столы
- `config/policy.yaml` — критичность, лимиты циклов, безопасность
- `cursor/subagents/` — роли агентов (18)
- `cursor/skills/` — стандарты (шаблоны и протоколы)
