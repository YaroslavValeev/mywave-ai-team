# Owner Decisions — MyWave AI-TEAM

Зафиксированные решения AGM/Owner по политикам и безопасности.

## Raw view — OFF по умолчанию

**Решение:** Redaction обязателен везде (Dashboard, reports, logs). Raw view разрешён только при:
- маркере `ALLOW_PII` в тексте задачи;
- запросе с `?raw=1`;
- при этом `scrub_secrets()` всегда применяется (токены/ключи никогда не показываются).

**Причина:** риск утечки через скрин/лог/проксирование. Консистентность политики "маскировать всегда".

## Dual auth (2-layer) — ON

**Решение:**
1. Внешний слой: Caddy BasicAuth (owner + пароль) для HTTPS-доступа.
2. Внутренний слой: X-API-Key (OWNER_API_KEY), прокидывается Caddy в app.

Браузеру не нужен ручной X-API-Key — Caddy добавляет его прозрачно. Оба слоя обязательны в production.

**Причина:** защита при компрометации одного из механизмов.

## Backups — daily 7/30

**Решение:** Ежедневные бэкапы БД. Retention:
- 7 daily (последние 7 дней);
- 4 weekly (по одному на воскресенье за последние 4 недели).

**Причина:** восстановление при потере данных, соответствие retention 90 дней для audit.
